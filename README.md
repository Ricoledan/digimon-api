# Digimon API

Run these commands to get started

# Run the program

```bash
deno run server.ts
```

# Run the program and watch for file changes

```bash
deno task dev
```

# Run the tests

```bash
deno test
```

# Run the benchmarks

```bash
deno bench --allow-net
```

# Check the list of available tasks

```bash
deno task
```

# Generate data

```bash
deno run -A ./scripts/generate_data.ts
```
